# Smart-Water-Data-Cleaning
# 🌊 智慧水务/高科技厂务：多维水质数据自动化清洗、趋势容错、分级预警与无人值守 ETL 系统 (V3.0)

## 1. 业务背景 (Business Context)
在高科技制造业（如半导体、新能源）的厂务物理运营以及市政“智慧水务”中，连续水质物联网传感器（COD/SS/TN/TP等）是系统的核心数据源。
在真实生产环境中，传感器常因网络瞬时抖动、零点漂移或局部水体不均（毛刺数据）产生误报。对于高价值、连续运转的工业产线，**因偶发数据异常而关停产线将造成灾难性损失**。
本项目不仅实现了多维脏数据的科学清洗，更设计了**“滑动窗口趋势容错”**与**“工业分级响应联动”**机制，并规划了**“无人值守的生产级数据管道（ETL Pipeline）”**，实现 IT 数据预警向 OT 生产现场安全运行的完美闭环。

## 2. 核心清洗与“软件容错”逻辑 (Core Cleaning & Fault Tolerance)
本系统支持 **COD、SS（悬浮物）、TN（总氮）、TP（总磷）** 四大核心水质指标的并发时序处理：
* 📈 **时序断点修复：** 采用 `interpolate(method='linear')` 线性插值法，科学还原传感器瞬时断线前后的连续水质变化轨迹，优于粗暴的常数填充。
* ⚙️ **物理规律纠偏：** 针对探头漂移产生的异常负值数据进行逻辑归零（`<=0` 归 0），符合现实物理规律。
* 🛡️ **环保工艺逻辑排雷：** 建立逻辑防火墙，自动识别并整行剔除“出水浓度 > 进水浓度”的严重失真假数据。
* 🔄 **瞬时毛刺容错（Sliding Window）：** 针对单一瞬间的指标尖峰（Spike），系统通过**滑动平均（Rolling Window）与连续次验证**进行智能去噪，避免单点波动造成的系统频繁误报。

## 3. 工业分级预警与 OT 响应机制 (Graded SOP & Industrial Linkage)
系统内置《城镇污水处理厂污染物排放标准》一级 A 排放阈值，并设计了符合工业生产抗风险要求的**三级联动响应预案（SOP）**：

| 预警级别 | 触发条件 | 软件判定 | 线下工业响应 SOP |
| :--- | :--- | :--- | :--- |
| 🟡 **黄色预警** | 单点/轻微超标 | 传感器波动或局部不均 | 系统弹窗提示，不干预工艺；自动派发巡检工单，进行物理清洗并人工采样比对。 |
| 🟠 **橙色预警** | 连续 3 小时超标，呈上升趋势 | 工艺系统发生轻微失控 | 触发自动化调控；系统联动药剂泵（PAC/PAM/碳源）加大投药量，调节曝气量。 |
| 🔴 **红色预警** | 持续极度超标，且工艺无法拦截 | 确认为严重环保安全风险 | **气动阀门自动切换，出水全面导入事故应急池缓冲**；前端生产产线不停机，保障工厂生产连续性。 |

## 4. 无人值守与生产级数据管道演进 (Enterprise ETL Pipeline Roadmap)
为解决手动运行脚本的局限性，本系统规划了向**生产级无人值守数据管道（ETL）**演进的系统架构：
[ 厂务/水务 IoT 数据库 ]
│ (InfluxDB / TimescaleDB)
▼
[ 动态时间窗口计算 (Python) ] ──► 自动计算上周一00:00至上周日23:00数据区间
│
▼
[ 核心 ETL 清洗与趋势容错 ] ──► 自动执行 V2.1 洗涤与多维遮罩算法
│
┌─────┴─────┐
▼ ▼
[写入洁净底表] [触发合规预警]
(Grafana/BI) │ (若触发 Orange/Red 预警)
▼
[ 自动化预警分发 ] ──► 邮件(SMTP) / 钉钉/企业微信机器人自动推送
* **动态时间窗口：** 脚本定期运行，自动根据当前系统时间向后推移 168 小时，动态锁定期初与期末时间戳，无需人工修改读取文件名。
* **定时调度（Workflow Scheduling）：** 部署至 Linux 环境，采用 **Crontab** 进行轻量级周级调度，或采用 **Apache Airflow / DolphinScheduler** 进行企业级工作流管理与运行状态监控。
* **多渠道预警分发：** 融合 Python `smtplib` 协议与 Webhook 接口，异常发生时自动将 `alert_log` 打包为附件发送至运维主管邮箱及企业通讯群。

## 5. 技术高光亮点 (Technical Highlights)
* **高度可扩展设计（Scalability）：** 将所有工艺标准配置于字典（Dictionary）中。未来若增加氨氮（NH3-N）等新指标，仅需在配置表增加一行，底层算法自动适配。
* **滑动计算与高效遮罩（Pandas Rolling & Masking）：** 使用 `df.rolling(window=3).mean()` 等 Pandas 矩阵化操作替代低效的按行循环，毫秒级实现时间序列趋势研判。

## 6. 文件结构 (Repository Structure)
* `generate_data_v2.py` : 模拟生成包含多维水质“脏数据”源文件。
* `clean_multi_water_quality_data.py` : 核心多指标自动化清洗、趋势容错与预警脚本。
* `cod_raw_data_202606.csv` : 一周 168 条测试集源文件。
* `clean_data_v2.csv` : 清洗去噪后、可直接对接数据大屏/数据库的干净底表。
* `alert_log_v2.csv` : 确认真实工艺失控的高价值超标预警日志。
